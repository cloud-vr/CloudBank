Why it was made?

Hi, I'm Vince Reyes and I work on Test Automation Projects. I wanted to explore the possibility of creating a generic test automation framework for transactional web applications.

To do that, I need a transactional web application to play with. I can't use websites like Amazon, Airbnb, and the like because I don't have control over the changes they do.

So my solution is to just make one myself. Thus, Cloud Bank.
